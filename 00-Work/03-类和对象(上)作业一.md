### 1. 请问在64位编译器下用 `sizeof(struct A)` 计算出的大小是多少

**📃 代码**

```c
struct A{
    
 long a1;
 short a2;
 int a3;
 int *a4;

};
```

**🔍 选项**

```
A.24

B.28

C.16

D.18
```

**✨ 解析**

```
答案：A
```
